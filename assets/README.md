# Ressources visuelles

Ce dossier contient les images et les icônes affichées par PILOTE.

## Icônes des navigateurs

Les icônes Edge, Chrome et Firefox proviennent de Tabler Icons. Elles sont enregistrées localement afin que l'interface ne charge aucune ressource depuis Internet.

Sources exactes :

- `https://raw.githubusercontent.com/tabler/tabler-icons/main/icons/outline/brand-edge.svg`
- `https://raw.githubusercontent.com/tabler/tabler-icons/main/icons/outline/brand-chrome.svg`
- `https://raw.githubusercontent.com/tabler/tabler-icons/main/icons/outline/brand-firefox.svg`

La licence MIT de Tabler Icons est conservée dans `BROWSER-ICONS-LICENSE.txt`.

Ces icônes servent uniquement à identifier les navigateurs compatibles avec chaque méthode d'export.
